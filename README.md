Finance Dashboard

A simple and responsive Finance Dashboard built using React.js and Tailwind CSS.
This application helps users track income, expenses, and overall balance with role-based access.

🚀 Features
     Dashboard Summary (Balance, Income, Expenses)
     Transaction Search (by category)
     Transaction Filter (Income / Expense)
     Role-Based UI (Admin & Viewer)
     Insights Section (Highest Spending, Summary)
     Transaction List with details
     Responsive Design
     Tech Stack


React.js (JavaScript)
Vite
Tailwind CSS
